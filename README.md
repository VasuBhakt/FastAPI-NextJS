# Inventory Management System

This is a basic full-stack inventory management system created with FastAPI in backend, SQLAlchemy for Database, and NextJS for frontend. 
This was made in an attempt to learn FastAPI from <a href="https://www.youtube.com/@Telusko">Telusko</a> and NextJS from <a href="https://www.youtube.com/@chaiaurcode">
Chai aur Code</a> Youtube channels

<a href="https://github.com/navinreddy20">Navin Sir</a> does an amazing job of introducing us to FastAPI and its features. He also utilises 
SQLAlchemy to give the prject a real-life feel! Definitely recommend the course for a quick intro to FastAPi, although if you want to go deeper,
you have to look out for things yourself.
